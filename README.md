# Project1
This is a simple Amazon clone I built to practice my frontend skills using HTML, CSS, and JavaScript. The main focus of this project was not to make a perfect replica, but to get hands-on experience with layouts, styling, and basic interactivity.
I tried to recreate the look and feel of Amazon’s homepage while working on things like responsive design, positioning, and DOM manipulation. It helped me understand how real-world websites are structured and how different components come together.
This project is purely for learning and practice — no backend, no real functionality like payments or authentication. Just frontend work to improve my fundamentals and get better at building UI.
